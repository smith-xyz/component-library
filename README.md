# Component Library

A ready to go React project with pre-equipped componenets that are customizable and reusable across projects.

## NavBar

| Components        | Description |
| ----------- | ----------- |
| NavBar   | Creates a container for nav bar. |
| NavHeader   | Creates a header icon for the nav bar. |
| NavItem   | Creates an item to use for navigation in nav bar. |
| NavDropdown   | Creates a dropdown list for navigation items. |

### API

| Prop        | Description |
| ----------- | ----------- |
| N/A     | N/A |

## Tables

| Component        | Description |
| ----------- | ----------- |
| DataTable   | Creates a container for nav bar. |

### API

| Prop        | Description |
| ----------- | ----------- |
| Headers     | An Array containing the top row column headers for the table. |
| rowData     | Data to populate rows, must be an Array of Objects. |
| dark     | Activates dark theme table. |
| size     | Sets size of table: "sm" "md" "lg" |
